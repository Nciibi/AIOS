## Purpose

The Constitutional Knowledge Repository is the authoritative store for the AIOS Constitution — its current text, its history, its interpretations, and its amendments. CKR makes the Constitution searchable, queryable, and machine-readable. Every entity should be able to answer: *what does the Constitution say about this?*

## Repository Contents

| Content | Description | Update Frequency | Source |
|---------|-------------|-----------------|--------|
| **Current Constitution** | The full ratified Constitution text | On amendment | CLS |
| **Law Definitions** | The 10 Laws with canonical text | Never (immutable) | CLS |
| **Amendment History** | All amendments with RFC references | On amendment | CLS → CRP |
| **Authoritative Interpretations** | Security Council rulings on constitutional questions | As issued | Security Council |
| **Physics Invariant Reference** | All Physics invariants cross-referenced to Laws | On Physics change | CLS |
| **Precedent Log** | Past constitutional challenges and their outcomes | On challenge | Security Council |
| **RFC Index** | All RFCs that affected the Constitution | On RFC approval | CRP |

## Query Interface

CKR exposes a query interface over ACF:

### Supported Queries

| Query | Description | Example |
|-------|-------------|---------|
| `getConstitution(version?)` | Retrieve current or historical Constitution text | `getConstitution("v1.0")` |
| `getLaw(law_id)` | Retrieve a single Law by its ID | `getLaw("law-8")` |
| `queryConstitution(query)` | Free-text search of Constitution text | `queryConstitution("verification")` |
| `getAmendmentHistory()` | List all amendments | `getAmendmentHistory()` |
| `getInterpretations(law_id)` | List Security Council interpretations of a Law | `getInterpretations("law-5")` |
| `resolveConstitutionalQuestion(question)` | Find relevant Laws, interpretations, and precedents | `resolveConstitutionalQuestion("Can an entity skip identity verification?")` |
| `checkConstitutionalCompliance(proposal)` | Check a proposal against the Constitution | `checkConstitutionalCompliance(rfc_draft)` |

### Query Authorization

| Query | Who Can Query |
|-------|--------------|
| `getConstitution` | Any entity |
| `getLaw` | Any entity |
| `queryConstitution` | Any entity |
| `getAmendmentHistory` | Any entity |
| `getInterpretations` | Any entity (unclassified), Security Council only (classified) |
| `resolveConstitutionalQuestion` | Any entity |
| `checkConstitutionalCompliance` | Any entity proposing a change |

## CKR Events

| Event Type | Produced When | Fields |
|-----------|--------------|--------|
| `CKR.ConstitutionUpdated` | Constitution text changes | new_version, rfc_id, changed_sections |
| `CKR.InterpretationIssued` | Security Council issues interpretation | law_id, interpretation_text, issuer_id |
| `CKR.PrecedentRecorded` | A constitutional challenge establishes precedent | challenge_id, ruling, basis |
| `CKR.ComplianceCheck` | A proposal passes constitutional compliance check | proposal_id, result, relevant_laws |
