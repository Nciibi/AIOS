## Purpose

The Autonomy Control Engine (ACE) enforces autonomy bounds at runtime. Every action an entity attempts passes through ACE before reaching the Security Council verification pipeline. ACE resolves the entity's current autonomy level into concrete capability bounds, filters actions against those bounds, and monitors compliance over time. ACE is the gatekeeper that ensures no entity exceeds its assigned autonomy without triggering escalation or denial.

### Compliance Monitor

The Compliance Monitor tracks every action an entity takes and maintains a compliance score that influences progression eligibility:

```

```

## Internal Interface

```typescript
interface ACE {
  // Bounds Resolution
  resolveBounds(entity_id: string): Promise<AutonomyBounds>
  getBounds(entity_id: string): Promise<AutonomyBounds>       // With caching
  invalidateCache(entity_id: string): void
  
  // Action Filtering
  checkBounds(action: Action, entity_id: string): Promise<FilterResult>
  filterAction(action: Action, entity_id: string): Promise<ActionFilterResult>
  
  // Compliance
  getCompliance(entity_id: string): Promise<ComplianceRecord>
  resetCompliance(entity_id: string): Promise<ComplianceRecord>
  applyDecay(entity_id: string, periods_elapsed: number): ComplianceRecord
}

interface BoundsResolver {
  resolve(entity_id: string, level: 0 | 1 | 2 | 3 | 4): Promise<AutonomyBounds>
  mergeBounds(base: AutonomyBounds, override: Partial<AutonomyBounds>): AutonomyBounds
  levelToActions(level: number): ActionScope[]
  levelToResources(level: number): ResourceBudget
}

interface ActionFilter {
  evaluate(action: Action, bounds: AutonomyBounds): FilterDecision
  checkScope(action: Action, allowed: ActionScope[]): ScopeResult
  checkResources(action: Action, budget: ResourceBudget): ResourceResult
  checkDelegation(action: Action, bounds: AutonomyBounds): DelegationResult
}

interface ComplianceMonitor {
  recordAction(record: ActionRecord): ComplianceRecord
  getScore(entity_id: string): number
  getHistory(entity_id: string, since: timestamp): ActionRecord[]
  calculateComplianceImpact(decision: "allow" | "deny" | "escalate"): number
}

type FilterDecision = {
  result: "allow" | "deny" | "escalate"
  matched_rules: string[]
  escalation_threshold?: EscalationThreshold
}

type FilterResult = {
  passed: boolean
  decision: FilterDecision
  bounds: AutonomyBounds
  requires_escalation: boolean
}
```

## Error Cases

| Override reference not found in ABE | `ACE_OVERRIDE_NOT_FOUND` | Resolve bounds without override; warn; continue |
| Delegation chain exceeds max depth | `ACE_DELEGATION_DEPTH_EXCEEDED` | Deny delegation action; return max depth allowed |
