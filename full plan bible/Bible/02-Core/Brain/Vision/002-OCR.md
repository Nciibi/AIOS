## Purpose

The OCR Engine extracts text from images with positional information, confidence scoring, language detection, and layout preservation. It handles printed text, handwriting, multi-language documents, barcodes, and structured layouts (tables, headings, paragraphs). OCR output feeds into higher-level processors â€” the Document Parser for field extraction and the Scene Describer for visible text inclusion.

Under VIS-006, OCR output below the minimum confidence threshold returns an error rather than hallucinated text. This ensures Sou never reasons about fabricated text content.

### Confidence Scoring

Confidence scoring operates at multiple granularities:

| Level | Scope | Enforcement |
|-------|-------|-------------|
| Per-block | Individual TextBlock | Each block has its own score |
| Per-table-cell | Individual TableCell | Each cell evaluated independently |
| Per-barcode | Individual Barcode | Decode confidence |
| Language | Per LanguageEntry | Detection reliability |
| Overall | Entire OCRResult | Average across all blocks |

### Minimum Confidence Enforcement (VIS-006)

### Layout Preservation

Layout preservation reconstructs the document structure:

| Feature | Detection Method | Output |
|---------|-----------------|--------|
| Headings | Font size, weight, position heuristic | `block_type: "heading"` |
| Paragraphs | Line grouping, indentation | Consecutive `block_type: "paragraph"` |
| Tables | Grid line detection, column alignment | `TableRegion` with cells |
| Lists | Bullet/number detection | `block_type: "list_item"` |
| Headers/Footers | Page position heuristic | Separate `block_type` values |

## Error Cases

| Condition | Error Code | Behavior |
|-----------|------------|----------|
| All text below min_confidence | `VIS_OCR_LOW_CONFIDENCE` | Return error; confidence scores included for debugging |
| No text detected at all | `VIS_NO_TEXT_FOUND` | Return empty result; not an error |
| Unsupported language detected | `VIS_OCR_LANGUAGE_UNSUPPORTED` | Fall back to generic OCR; log warning |
| Barcode decode failed | `VIS_BARCODE_DECODE_FAILED` | Skip barcode, continue text extraction |
| Image too blurry for OCR | `VIS_OCR_IMAGE_TOO_BLURRY` | Return error; suggest enhancement or recapture |
| Layout analysis failed | `VIS_OCR_LAYOUT_FAILED` | Return flat text blocks without structure |
| Handwriting detector failure | `VIS_OCR_HANDWRITING_FAILED` | Assume printed text, log warning |
| Provider returned empty result | `VIS_OCR_EMPTY_RESULT` | Retry once with different provider; log |
| Language hint mismatch | `VIS_OCR_LANGUAGE_MISMATCH` | Log warning; continue with detected language |
| OCR request exceeds timeout | `VIS_OCR_TIMEOUT` | Return partial results with timeout flag |
