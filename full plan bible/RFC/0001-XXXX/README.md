# RFC-0001

This directory is a placeholder for the first RFC. When proposing a new RFC:

1. Copy `Templates/RFC-template.md` into a new directory named `NNNN-Short-Name/`
2. Rename `XXXX` to the RFC number and `0001-XXXX` to the actual name
3. Fill in the template
4. Submit through CRP (see `000-RFC-Process.md`)

Example: `0001-ACF-Topic-Naming/`
